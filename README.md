# Fantasy Map Simulator

WorldBox 風のファンタジーマップシミュレーター。プロシージャル生成された大陸マップ（640×640 タイル）を、神の力（地形・生命・文明・災害ブラシ）で編集し、**気候・植生・生物・炎・文明（王国）**が相互に影響しあう生きたシミュレーションとして動きます。

バニラ JavaScript + HTML5 Canvas 製。**ビルド不要・依存ゼロ**。PC・スマホ・タブレット対応（レスポンシブ + 高DPI）。

## 主な機能

### シミュレーション
- **気候・四季・昼夜** — 春夏秋冬が時計で巡り、季節が植生の成長・炎の延焼しやすさ・体感気温に影響。**昼夜サイクル**で世界の明暗が変わり、夜は**都市が灯る**。
- **バイオーム** — 標高・湿度・**温度（緯度＋標高）**で分類。砂漠・サバンナ・湿地・ツンドラ・ジャングルなどを生成し、湿った高地から海へ**川**が流れる。
- **植生・生態系** — タイルごとの植生密度(fertility)が季節に応じて再成長。育てば**森林化**、枯れれば後退、焼け地は徐々に回復。草食動物の採食で消費される（ローリング走査で軽量）。
- **生物・生命** — 草食／肉食動物が採食・捕食・繁殖・寿命/餓死。**遺伝子（体格）**が速度・燃費・見た目に影響し、繁殖で変異を継承。**渇き**で水辺を求めて移動。植生を介した密度依存で個体数が自律安定。
- **炎の延焼** — 可燃地形に着火すると確率的に延焼し、燃え尽きて焼け地に。季節・乾燥（低 fertility）で延焼しやすさが変わる。
- **文明・王国（人間主導 / ボトムアップ）** — まず**人間（放浪者）を撒く**と、彼らが歩いて仲間と肥沃地に集まり、**自分たちで国を興す**（「国を置けば人が湧く」のではなく、人が先で国が後）。建国後は人間が未開の陸地を歩いて確保し、領土が創発する。**他国の領土は歩いただけでは奪われず**、征服は兵士が自国に隣接する前線でのみ行う。各国に固有名。**人口＝実在する人間エージェントの数**、放浪者は別に集計。
- **人間AI（欲求・職業・社会）** — 各人間は AI を積んだ自律エージェント:
  - **欲求**: 空腹（可食地で採食・餓死あり）／社交（同胞の近くに集う）／加齢（成人→老い→寿命）。優先度で行動を決める。
  - **職業(role)**: 開拓者（領土拡張・新集落建設）／農民（土地を耕して肥沃化）／建築家（集落の発展度を上げる）／兵士（敵と戦う）。役割は帽子の色で区別。
  - **社会関係**: 氏族(血統)と王国(in-group)で結びつき、**成人ペアで子をなし家族が増える**。他王国は敵(out-group)で兵士が交戦。確保した土地（食料）に応じて人口が増え、遠征した人間が新集落を興して文明が広がる。
  - **放浪者(無所属)の判断**: 近くに既存国があれば**その国へ加入**し、無ければ仲間と肥沃地で**自ら建国**、それも無ければ定住先を探して移動する（目的を持って行動）。
  - **定住と職住近接**: 各人間は**所属する町(home)**を持ち、農民は町の近くに**自分の耕作地**を構えて通い、建築家は町に詰める（開拓者・兵士のみ遠出）。慣性のある滑らかな移動で町の周りに自然に集まる。
  - **移民**: 国が過密で食料も乏しいと、住民は町を捨て**流民となって新天地を探す**（移民の流れが創発）。
  - **役割ごとの行動圏**: 農民は町を囲む農地に散らばって耕し、建築家は町を整え、兵士は領内を警邏（戦時は前線へ）、開拓者は辺境を開く。各自が別々の区画を巡回するので中心に固まらず、領土全体が賑わう。
  - **征服と難民**: 戦争で都市が陥落すると**都市は勝者のものになり**、最後の都市を失うと国家は崩壊（遷都あり）。国が滅ぶと住民は死なず**難民**となって他国へ加入・再建を目指す。
  - **現実的な領土ルール**: 領土は都市から**地続きにのみ拡張**（斑点・飛び地が出ない連続した版図）。各都市の**支配圏（半径）**を超えた辺境は手放し、無人・無領土の国は消滅、囲まれた飛び地は吸収される。版図は都市網に応じて伸縮する（ローリング走査で軽量）。
  - **効率化**: 近傍探索は一様グリッドで O(1)。**重い意思決定・探索・征服判定はすべて数ティックに1回**（位相分散）に集約し、毎ティックは加齢・採食・足下確保・移動など軽量処理のみ。兵士の戦闘は思考時にキャッシュした標的を使い毎ティックの探索を排除。**1500人規模でも約0.5ms/tick・60 FPS**。
- **国家・外交システム** — 各国に**統治者・政体**を持ち、国境で接触した国同士に**外交関係**が生まれる。関係に応じて**開戦・講和・同盟**が起こり、戦争中の国だけが前線で領土を奪い合う（平時は国境を尊重）。**諸国パネル**で各国の人口・都市・領土・交戦/同盟を一覧でき、クリックで首都へジャンプ。

- **天候** — 雲が大陸を流れ、**雨**で湿度・植生を潤し、**落雷**で可燃地に着火する（雨は延焼を鎮める）。季節で雨量・落雷頻度が変化（春は雨がち、夏は雷雨）。
- **技術・時代・宗教** — 各国は都市数・人口・富に応じて**技術**が伸び、**石器→青銅器→鉄器→古典→中世→啓蒙**と**時代**が進む。各国に**宗教**があり、国力の大きい国の宗教が弱い国へ広まる。
- **指導者の性格** — 各国の統治者は**好戦的／温厚／商才／敬虔／賢明**のいずれかを持ち、開戦・同盟・交易・技術・宗教伝播のしやすさを変える。
- **戦争（軍事力で決まる）** — 国の**軍事力**（兵士数×技術）の比で前線の征服速度が決まり、強国が弱国を押す。**戦争が長引く・劣勢になると講和**しやすくなり、都市陥落で領土が移り、最後の都市を失えば国家は崩壊する。**異教**どうしは戦になりやすく、**同盟国は参戦（呼びかけ）**してブロック戦争になる。
- **経済・交易** — 領土から収入が生まれ、**戦争でない隣国とは交易**して双方が富み少し友好も深まる（同盟は倍、商才で増す）。同盟国は**交易路**（金色の線）で結ばれる。富は技術を加速する。
- **宗教と外交** — **同教は同盟・交易、異教は緊張・戦争**に傾く（宗教ブロックが創発）。
- **不満と反乱** — 戦争・過密・貧困で**不満**が高まり、限界を超えると地方都市が**独立して新国家**になる（独立戦争・国家分裂が創発）。

### 描画・パフォーマンス（追加）
- **国境線**（所有が変わる辺を国の色で縁取り、領土をはっきり可視化）、**街道**（首都と都市を結ぶ線）、**国名ラベル**（首都の上に表示）、**タイル詳細インスペクタ**（カーソル位置のバイオーム・標高・気温・湿度・植生・所属国/宗教）。
- **ヘルプ＆凡例**（右下「?」）で操作方法とバイオームの色見本を表示。

### デバイス最適化
- 端末を判定し、**スマホ／タブレット／PC** ごとに地図サイズ・エージェント上限・初期ズーム・UI レイアウトを自動調整（スマホは軽量プロファイル＋折りたたみパネル）。
- **災害** — 地震・隕石（クレーター＋火災）・洪水・疫病。

### 描画・パフォーマンス
- **ピクセルアート描写** — 近景では草食（鹿/羊風）・肉食（狼/狐風）を**ドット絵スプライト**で描画。進行方向で左右反転し、体格は遺伝子で変化。
- **人間のドット絵** — 市民は頭・髪・胴（国色）・腕・脚・目・役割の帽子を持つ**人型ドット絵**で描画（向きで顔の向きも変化）。
- **街並み** — 集落は**家々のドット絵**が立ち並び、首都には**砦**がそびえる（発展度で軒数増）。**街道**は路面付きの道として描画。農民は自分の畑に通い、人々が家と畑の間を行き交う。
- **2段階 LOD 描画** — 遠景は種別ごとに一括 fillRect、近景のみピクセルスプライト。**15,000体でも 60 FPS**を維持。
- **大規模対応** — マップ 640×640、生物上限 16,000・市民 700・王国 96。

### UI
- **上部ステータスバー** — タイトルと季節時計（🌸春 / ☀️夏 / 🍁秋 / ❄️冬）。
- **カテゴリ別ツールバー** — 地形 / 生命 / 文明 / 災害 にグループ分け。
- **ライブ統計 HUD** — 個体数・草食/肉食・王国・人口・延焼・FPS と、個体数推移のスパークライン。
- **ミニマップ** — 全体地形＋領土を縮小表示。ビューポート矩形、クリック/ドラッグでジャンプ。
- **シミュレーション制御** — 一時停止／再生と速度（0.5〜4倍）。固定タイムステップで描画から分離。

## Web で遊ぶ（GitHub Pages）

`main` または開発ブランチへ push すると、単一の CI ワークフロー（`.github/workflows/ci.yml`）が **テスト → `gh-pages` へデプロイ → 過去の実行ログを自動削除** まで全自動で実行します（Actions タブが散らからないよう、起動のたびに本実行以外の古い run を消去）。

公開URL（Pages 有効化後）:

```
https://nomixio260-a11y.github.io/project2/
```

**初回のみ**: リポジトリの **Settings → Pages → Build and deployment → Source** を **「Deploy from a branch」** にし、ブランチを **`gh-pages` / `(root)`** に設定してください。以降は `main`（または作業ブランチ）へ push するたびに `gh-pages` が自動更新されます。

## ローカルで起動

### 1. ただ開くだけ
`index.html` をブラウザにドラッグ＆ドロップ、またはダブルクリックするだけで動きます（`file://` で動作）。

## 神の力（ツール）

地形（1〜8）:

- **隆起 / 沈下** — 土地を盛り上げる・沈める
- **水 / 砂 / 草原 / 森 / 山** — 地形を直接描く
- **着火（8）** — 可燃地形に延焼する炎を放つ（不燃地は即焼け地）

生命・文明:

- **草食（9） / 肉食（0）** — 生物をスポーン
- **建国（K）** — その地に王国を興す

シミュレーション制御: **一時停止（P）** と **速度（0.5×〜4×）**。

## アーキテクチャ

`window.Game` 名前空間にモジュールがぶら下がる構成。`index.html` がスクリプトを依存順にロードします（no-build 制約のため ES Modules ではなくクラシック `<script>` を使用）。

```
js/
├── core/      namespace, constants, utils
├── math/      noise (Simplex + fBm)
├── world/     world(データ), tile(分類), worldgen(生成), entities(生物ストア)
├── render/    camera, renderer
├── tools/     brush, godpowers
├── input/     input
├── ui/        toolbar
├── systems/   creatures, fire, civ（engine.systems に登録）
├── engine/    engine（固定タイムステップのゲームループ）
└── main.js    起動
```

各シミュレーションは `engine.systems` に登録され、固定タイムステップの `tick(world)`
（一時停止・速度に従う）と毎フレームの `update(dt, world)`（アニメーション）を持ちます。
レンダラは地形（オフスクリーン）→ 領土オーバーレイ → 炎 → 生物 の順に毎フレーム合成します。

## テスト

```bash
npm test            # ユニットテスト（コアロジック / 依存ゼロ）
npm run test:browser   # 実ブラウザ(Puppeteer)スモークテスト
npm run test:all       # 全テスト
```

- `tests/unit.test.js` — ノイズ・地形生成・座標変換などのロジック検証（DOM不要）
- `tests/browser.test.js` — Puppeteer で実描画・編集・タッチ・レスポンシブを検証

CI（`.github/workflows/test.yml`）で push / PR ごとに自動実行されます。
